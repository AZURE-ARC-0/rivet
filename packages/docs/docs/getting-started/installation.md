# Installation

## Releases

Not yet available.

## Building from Source

### Prerequisites

- Rust (use [rustup](https://rustup.rs/))
- node 16.18+
- yarn

### Install

```bash
git clone https://github.com/Ironclad/rivet.git
cd rivet
yarn
```

### Build & Run

```bash
yarn dev
```

This will build and run the application in development mode.
